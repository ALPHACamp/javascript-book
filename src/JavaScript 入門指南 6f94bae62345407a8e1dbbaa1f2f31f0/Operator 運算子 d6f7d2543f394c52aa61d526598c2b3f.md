# Operator 運算子

運算子 (operator) 可以對 value 做處理，並且回傳新的 value，我們將會介紹以下運算子：

- 算術運算子（arithmetic operators）
- 賦值運算子（assignment operators）
- 比較運算子（comparison operators）
- 邏輯運算子 （logical operators）
